Project Name: THE SUIT

Objective:
Build a luxury room booking platform for a single property (for now), inspired by Airbnb. The platform should feel premium, smooth, and highly responsive, with a strong focus on UI animations and mobile usability.

IMPORTANT:
This version supports only ONE property. Users interact directly with rooms. The system must still be structured in a way that allows adding multiple properties later without major refactoring.

---
